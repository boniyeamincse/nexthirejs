## 11. Bootstrap Requirements

Implement `src/main.ts` with the following behavior.

### 11.1 Application Configuration

- Create the Nest application.
- Use `ConfigService`.
- Use the configured host and port.
- Enable graceful shutdown hooks.
- Use `NestExpressApplication` if required for request-size configuration.

### 11.2 Global Prefix

Default:

```text
api
```

Result:

```text
/api/v1
```

The prefix must come from:

```text
API_GLOBAL_PREFIX
```

### 11.3 URI Versioning

Enable URI versioning with default version from:

```text
API_DEFAULT_VERSION
```

Use NestJS URI versioning.

Expected versioned endpoints:

```text
GET /api/v1
GET /api/v1/health
```

Swagger itself may remain at:

```text
/api/docs
```

### 11.4 Validation Pipe

Configure a global `ValidationPipe` with:

```typescript
{
  whitelist: true,
  forbidNonWhitelisted: true,
  transform: true,
  transformOptions: {
    enableImplicitConversion: false,
  },
}
```

Do not enable implicit conversion globally.

### 11.5 Security Headers

Use Helmet globally:

```typescript
app.use(helmet());
```

Do not disable security headers without a documented reason.

### 11.6 Request Body Limit

Use the configured body limit:

```text
API_BODY_LIMIT
```

Apply it to JSON and URL-encoded bodies.

Use Express middleware safely.

### 11.7 CORS

Read allowed origins from:

```text
API_CORS_ORIGINS
```

Rules:

- Accept comma-separated origins.
- Trim whitespace.
- Reject arbitrary origins not in the allowlist.
- Allow requests without an `Origin` header for CLI, server-to-server, health checks, and tests.
- Enable credentials.
- Do not use `origin: '*'` together with credentials.
- Allow common methods:
  - GET
  - HEAD
  - POST
  - PUT
  - PATCH
  - DELETE
  - OPTIONS
- Allow common headers:
  - Content-Type
  - Authorization
  - X-Request-ID
  - Idempotency-Key

If the allowlist is empty in development, fail safely and document the behavior. Do not silently allow every origin.

### 11.8 Startup Log

Log the final local URL and Swagger URL using NestJS `Logger`.

Do not print secrets or the complete environment.
